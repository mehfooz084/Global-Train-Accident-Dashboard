
# Global Train Accident Analysis Dashboard (2000–2024)

## Overview
This Power BI dashboard analyzes global train accident trends, severity patterns, accident causes, fatalities, injuries, and geographical hotspots between 2000 and 2024.

## Tools Used
- Power BI
- DAX
- Python (EDA)
- Data Visualization

### Key Insights
- Total recorded accidents: **412**
- Total injured: **22K**
- Total deaths: **6K**
- Fatal accident rate: **50%**
- Accident frequency increased after 2010, peaking around 2018–2020.
- Severe accidents are concentrated in specific geographic regions rather than evenly distributed globally.

### Key Insights
- Infrastructure Failure and Human Error account for nearly **80% of total accidents**.
- Passenger trains experienced the highest number of accidents.
- Derailments resulted in the highest average fatalities.
- India recorded the highest average deaths among analyzed countries.
- Derailments and collisions contributed most to casualties.

### Page 3: Insights & Recommendations

<img width="1227" height="688" alt="Screenshot 2026-06-01 191913" src="https://github.com/user-attachments/assets/67bfbe1f-ad3b-4376-99b3-9ac7aace878f" />

### Major Findings
- Infrastructure Failure and Human Error are the dominant risk factors.
- Fatal accidents increased noticeably after 2010.
- High-risk regions show clustering of severe incidents.
- Injuries consistently exceed fatalities, indicating opportunities for improved emergency response.

### Recommendations
- Strengthen infrastructure inspection and maintenance programs.
- Reduce human error through automation and real-time monitoring.
- Improve passenger train safety systems.
- Implement predictive maintenance and derailment prevention measures.
- Enhance post-accident emergency response capabilities.
